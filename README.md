### Welcome to Meridix-WebAPI-JS
This is a library that can be used to sign API requests against Meridix WebAPI.

### Installation
The package is available through NPM

    npm install meridix-webapi --save
    
### Dependencies
CryptoJS
    
### Build
To build the package

    npm run build
